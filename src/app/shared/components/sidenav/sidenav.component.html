About this demo

This is a demonstration of a dynamic side navigation with Angular. The application showcases different modules named after parts of a house for educational purposes.

The main goal is to provide a clear example of how to implement a side navigation with dynamic width. Each section, such as Garage, Bedroom, and Kitchen, is represented as an expansion panel in the side navigation.

The main content area in the middle is resizable and can be toggled to fullscreen. You can navigate between different sections using the buttons provided or by clicking on the respective panel titles.
